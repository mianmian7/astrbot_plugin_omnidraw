## 本插件支持：
- **文生图**
- **图生图（改图）**
- **自拍模式（一次性上传参考图+人设参考+改图）**
- **视频生成（长视频异步渲染）**
- **动态模型热切换**
- **副脑功能（使用专业内置提示词（与人设隔离）自动优化提示词 一句简短的话也能生成高质量图片 可选择开关）**
- **一次请求多张图（需开启副脑功能）**
- **图生图/改图预设功能**

## 🌟 核心特性

- **🤖 自动选择文生图/图生图**：支持有参考图自动选择图生图，无参考图自动选择文生图
- **🚀 工业级异步视频引擎**：支持 Veo、Kling (可灵) 等长耗时视频模型。采用“提交-挂机-推送”架构，视频生成期间大模型不阻塞，可继续陪聊。
- **🛡️ QQ 防盗链完美突破**：自动下载 QQ 原始图片到本地持久化，并转化为 Base64 注入 API 协议。彻底解决 `400 Bad Request` 和图片无法读取的问题。
- **🔄 模型热切换系统**：无需重启！通过 `/切换模型` 指令实时在多个绘图/视频模型间无缝流转。
- **⚡ 极致性能监控**：在后台日志中精准输出 API 响应时间与发送总耗时，助你掌握每一毫秒。
- **🎭 拟真交互设计**：支持固定人设（Base Prompt + 参考图）自拍。大模型调用工具后自动移交回复权，实现图文一体的拟真对话。
- **🛠️ 智能接口适配**：视频生成自动尝试 `Chat/Completions`、`Videos`、`Images` 三大端点，适配市面 99% 的中转站。

## 🎮 指令说明

| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `/画 [提示词]` | 基础绘图（支持携带参考图） | `/画 赛博朋克猫咪` |
| `/自拍 [动作]` | 以固定人设生成自拍照片 | `/自拍 在海边跑步` |
| `/视频 [提示词]` | 异步生成视频（支持携带多张参考图） | `/视频 两个人在森林里跳舞` |
| `/切换模型 [目标]` | 列出当前节点可用的所有模型 | `/切换模型 画图` |
| `/切换模型 [目标] [序号/名称]` | 切换到指定模型，目标支持画图/自拍/视频 | `/切换模型 画图 2` |
| `/万象帮助` | 查看插件完整帮助文档 | `/万象帮助` |

## 🛠️ 配置指南

插件已完美适配 AstrBot 可视化控制面板。进入 WebUI 插件设置页面即可看到：

1.  **权限控制**：在 `allowed_users` 中填入管理员 QQ 号，限制调用权限。
2.  **图片/视频节点**：支持配置多个 Provider。可以在可视化页面的模型池中逐个添加模型；旧配置里在 `model` 栏位使用 **英文逗号** 分隔（如：`dall-e-3,flux-1.1,midjourney`）也会被自动归一化为模型池。
3.  **人设配置**：
    - `persona_name`: 助理的称呼。
    - `persona_base_prompt`: 锁定助理外貌的关键词。
    - `persona_ref_image`: 上传一张形象参考图。

## 💡 使用小贴士

- **视频多图参考**：使用 `/视频` 指令时，你可以一次性发送 2-3 张图片，插件会自动将其全部作为视频生成的视觉参考。
- **长连接配置**：由于视频模型通常需要 2-5 分钟，请确保在 WebUI 设置中将视频节点的 `timeout` 设置为 `300` 或更高。

---

> **注意：** 本插件需要 `aiohttp` 支持，已在 `requirements.txt` 中声明。若使用 `v1/chat/completions` 生成视频，请确保你的中转站模型支持该端点的视频输出。
